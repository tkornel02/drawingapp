# Drawingapp
Simple Paint-like drawing app in Java with various tools
## Launch app
**Clone repo**
```bash
git clone https://github.com/tkornel02/drawingapp.git
cd drawingapp
```
**Compile**
```bash
javac -d ./out -cp .\src .\src\Main.java
```
**Run**

```bash
java -cp ./out Main
```
## Functions
- Draw line/rectangle/circle/curve
- Eraser function
- Clear Canvas
- Save as PNG
- Change color
